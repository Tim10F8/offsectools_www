Weekly newsletter n°29

offsec.tools

A vast collection of security tools for bug bounty, pentest and red teaming
offsec.tools is a vast listing of security tools designed to help pentesters and bug hunters in their daily task. The list is organized by tags and provide a quick search engine. The list is feeded by the author and the community. Anyone can add a tool and be listed as a contributor, feel free to check the GitHub repository.


You may have noticed a few changes in the design recently as I'm trying to make the site more pleasant to use and more efficient at the same time. Last week I moved many assets on AWS to improve the loading.
The template for contributions has been updated a bit to make it easier to parse, so please take care to use the new format!

Happy hunting!
Go to offsec.tools
